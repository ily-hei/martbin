Esta carpeta contiene las imágenes utilizadas en el sitio web del tanque de basura inteligente.

1. **product-hero.jpg** - Imagen principal del producto (1200x800px)
2. **product-features.jpg** - Imagen de características del producto (800x600px)

- Todas las imágenes deben estar optimizadas para web
- Usar formatos modernos como WebP cuando sea posible
- Incluir versiones de alta resolución para pantallas Retina
- Las imágenes deben ser responsivas y adaptarse a diferentes tamaños de pantalla
